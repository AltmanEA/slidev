# Работа ИИ агента

Все действия (создание или редактирование файла, класса, функции и др.) ИИ должен согласовать с пользователем.
При возникновении проблем предложить пользователю выбрать варианты решения, из которых пользователь выберет лучший или предложит свой.
Для решения сложных задач, требующих нескольких действий нужно разбить задачу на несколько этапов и согласовать этот план с пользователем.

# Среда разработки

Агент работает в vscode, плагин koda. 
Система windows, использовать в терминале команды powershell.
Аргументы команд (например, пути) нужно заключать в кавычки.

# Работа над проектом

Документация находится в папке docs/, исходные коды в src/, тесты в src/test/.
Перед выполнением каждой задачи нужно ознакомиться с документацией.
После выполнения каждой задачи нужно написать тесты для новой функциональности и проверить работу.
После успешного выполнения тестов нужно обновить документацию.
