## Для разработчиков

### Быстрый старт
1. Клонируйте репозиторий
2. Установите зависимости: `pnpm install`
3. Скомпилируйте проект: `pnpm run compile`
4. Запустите тесты: `pnpm run test:unit`

### Разработка
- **Язык**: TypeScript
- **Тестовый фреймворк**: Mocha
- **Сборщик**: tsc (TypeScript Compiler)
- **Пакетный менеджер**: pnpm

### Основные команды
```bash
pnpm run compile    # Компиляция TypeScript
pnpm run test:unit  # Запуск unit-тестов
pnpm run lint       # Проверка кода линтером
```

### Структура проекта
```
src/
├── services/        # Основные сервисы
├── types/          # TypeScript типы
├── test/           # Тесты
└── extension.ts    # Точка входа расширения
```
